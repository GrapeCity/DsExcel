Excel Input&Output Demo

This example demonstrates how to use GcExcel as server-side spreadsheet model, and use Spread.Sheets as the front-end viewer and editor.

  • GcExcel can import an excel file and export to ssjson format, then transport the ssjson to client-side.
  • Spread.Sheets client-side can receive and load the ssjson from server-side.
  • You can view the content of excel file through Spread.Sheets.
  • You can also make changes to the content, and send the whole document as ssjson to GcExcel server-side.
  • GcExcel server-side load the ssjson and saves to a new excel file, then you can download the modified excel file.
